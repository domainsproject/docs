## How to contribute

There are many ways to help:

- Submit a pull request
- Improve the documentation
- Review open PRs from others
- Suggest new features or domains
- Help triage issues and questions

If you’ve made a contribution and don’t see your name listed here, let us know and we’ll happily fix that.
